# CryptoVerse

## Live Demo

https://crypto-verse-lobnor.vercel.app/

## Run Locally

Clone the project
```bash
  git clone https://github.com/lobnor2/Crypto-verse.git
```

Go to the project directory

```bash
  cd Crypto-verse
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
